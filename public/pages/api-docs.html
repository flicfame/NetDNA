NetDNA API Reference
REST API v1 · All endpoints require authentication unless noted
🔐 Authentication

All API requests must include a JWT Bearer token in the Authorization header.
Obtain a token via POST /api/v1/auth/login with your credentials.

Header format: Authorization: Bearer <token>

Authentication
POST /api/v1/auth/login Authenticate and receive JWT token
FieldTypeDescription
usernamerequiredstringYour NetDNA username
passwordrequiredstringYour password
Request
curl -X POST http://localhost:8000/api/v1/auth/login \ -H "Content-Type: application/json" \ -d '{"username":"admin","password":"netdna2024"}'
Response
{ "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "token_type": "bearer", "user": { "username": "admin", "role": "admin", "permissions": ["view_dashboard", "quarantine", ...] } }
GET /api/v1/auth/me Get current user info 🔒 Auth required
curl http://localhost:8000/api/v1/auth/me \ -H "Authorization: Bearer <token>"
